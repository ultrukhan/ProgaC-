# ProgaC-
---
## Жоский репозиторій для навчання
### Щоб стягнути репозиторій на комп
```
git clone https://github.com/ultrukhan/ProgaC-.git
```
### Щоб стягнути останні зміни
```
git pull
```
### Щоб створити нову гілку від існуючої і одразу переключитись на неї
```
git checkout -b назва_нової_гілки
```
### Щоб зберегти нові додані файли в репозиторій 
```
git add .
git commit -m "Зроблено (якісь) зміни"
git push origin main
```
